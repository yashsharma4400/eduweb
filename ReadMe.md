# EduWeb - Online Learning Platform

A modern, responsive educational website built with HTML, CSS, and JavaScript. EduWeb provides an interactive platform for browsing and enrolling in online courses with a user-friendly interface.

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **User Authentication**: Login and registration functionality for student accounts
- **Course Catalog**: Browse and explore various online courses
- **About Section**: Learn more about the platform and its mission
- **Contact Page**: Easy way for users to get in touch
- **Interactive UI**: Smooth animations and transitions for enhanced user experience
- **Swiper Carousel**: Dynamic course carousel for featured courses

## Technologies Used

- **HTML5**: Semantic markup for structure
- **CSS3**: Custom styling with responsive design and animations
- **JavaScript**: Vanilla JS for interactivity and form handling
- **Font Awesome**: Icon library for UI elements
- **Swiper.js**: Touch slider for course carousel
- **Google Fonts**: Nunito font family for typography

## Installation & Setup

1. **Clone or Download** the project files
2. **Open in Browser**: Simply open `home.html` in your web browser
   - No server required for basic functionality
   - Works offline for static content

## Usage

1. Navigate to the home page to see featured courses
2. Click "Courses" to view all available courses
3. Use "About" section to learn more
4. "Contact" page to send inquiries
5. Click the user icon to login or register
